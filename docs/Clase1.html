Creacin de variables

Comencemos creando variables numricas con un nico valor

x <- 5 # En este caso, x tendr el valor de 5
y <- -3 # En este caso, y tendr el valor de -3

Si queremos crear variables numricas con varios valores, tendremos que crear un vector utilizando el comando c(), en donde cada valor estar separa por coma.

Peso <- c(12.5, 23, 12.6, 18, 7, 18.3, 23, 40, 21,8)

Observe que en este caso las comas se parados los valores, y los puntos solo se utilizan para las cifras decimales. La variable Peso tendr 10 observaciones.

Otra forma de crear un vector es utilizando el signo : entre los valores del inicio y final.

Altura <- c(1:1000)

En este caso la variable Altura tendr 1000 observaciones

Ahora, cmo podemos crear una variable con nmeros aleatorios?, como podemos crear una variable con distribucin normal?

Variable con numeros alatorios

Primero podramos utilizar el comando sample() para crear nmeros aleatorios sin una distribucin en especifica, dando el nmero de inicio y final, y la cantidad de observaciones que deseamos.

Aleatorios <- sample(1:1000, size = 50)

En este ejemplo observe que, 1 y 1000 es el rango entre el cual estarn 50 observaciones.

Variable con distribucion normal

Podemos crear variables con una distribucin normal utilizando el comando rnorm(), en este caso tendremos que especificar la media y la desviacin estndar.

Dnormal <- rnorm(50, mean = 30, sd = 2.5)

Para este ejemplo, hemos creado 50 observaciones, con una media de 30 y una desviacin estndar de 2.5

El siguiente paso es la creacin de variables categoras, variables que incluyen categorias. Para crear una variable de categoras tendremos que colocar cada observacin entre comillas " "

Variabley <- "y"

En este ejemplo la variabley solo tiene la observacin y. Si quermos ms observaciones podemos utilizar el comando c().

Seccion <- c("1A", "1B", "3A", "2B", "2A", "3A")

2) Normalidad y homogeneidad en la variable respuesta.

Un supuesto de algunas pruebas estadsticas es que la variable respuesta sigue una distribucin normal. Para verificar este supuesto, se puede utilizar una prueba de normalidad, como la prueba de Shapiro-Wilk. La hiptesis nula de esta prueba establece que los datos se distribuyen normalmente. Si el valor p es menor que 0.05 (p < 0.05), se rechaza la hiptesis nula.

shapiro.test(cyanistes$depth)
## 
##  Shapiro-Wilk normality test
## 
## data:  cyanistes$depth
## W = 0.89872, p-value < 2.2e-16

La homogeneidad de varianzas es otro supuesto importante en muchas pruebas estadsticas. Para visualizar la homogeneidad de varianzas en una variable categrica, se puede utilizar un grfico de caja. Sin embargo, un grfico de dispersin es ms adecuado para visualizar la homogeneidad en variables numricas.

library(ggplot2)

3) Exceso de ceros en la variable respuesta.

Los ceros no pueden ser omitidos de un conjunto de datos; sin embargo, un exceso de ceros en la variable respuesta, conocido como inflacin de ceros, puede causar problemas en los anlisis. Una forma de evaluar la inflacin de ceros es calculando el porcentaje de ceros en la variable:

sum(cyanistes$depth == 0, na.rm = TRUE) * 100 / nrow(cyanistes)
## [1] 0

En este caso, no tenemos problemas con los ceros. Sin embargo, de presentarse, tendramos que buscar alternativas para manejarlos, lo cual abordaremos ms adelante.

4) Multicolinealidad entre variables predictoras.

La multicolinealidad entre variables predictoras se refiere a una situacin en la que dos o ms variables independientes en un modelo de regresin estn altamente correlacionadas entre s. Esto puede dificultar la estimacin de los coeficientes del modelo, ya que hace que sea complicado determinar el efecto individual de cada variable en la variable dependiente. La multicolinealidad puede inflar los errores estndar de los coeficientes, lo que lleva a inferencias estadsticas poco confiables, como valores p engaosos y una disminucin en la capacidad predictiva del modelo.

5) Relaciones entre variables respuesta y predictoras.

Las relaciones entre variables respuesta y predictoras se refieren a cmo la variable dependiente (respuesta) se ve afectada por las variables independientes (predictoras) en un conjunto de datos. Examinar estas relaciones es fundamental en la exploracin de datos porque permite identificar patrones, tendencias y correlaciones que pueden influir en el comportamiento de la variable respuesta. Comprender estas relaciones ayuda a construir modelos predictivos ms precisos, a formular hiptesis y a guiar la interpretacin de resultados, facilitando as la toma de decisiones informadas en el anlisis estadstico y en la investigacin.

6) Independencia en las observaciones de la variable respuesta.

La independencia en las observaciones de la variable respuesta se refiere a la suposicin de que cada observacin en el conjunto de datos es independiente de las dems. Esto significa que el valor de una observacin no debe influir en el valor de otra. La independencia es crucial en el anlisis estadstico porque garantiza que las inferencias y los resultados sean vlidos. Si las observaciones no son independientes, puede haber una correlacin no reconocida entre ellas, lo que puede conducir a errores en la estimacin de parmetros, aumento de la varianza de los errores y, en ltima instancia, a conclusiones incorrectas sobre las relaciones entre variables. Por lo tanto, verificar la independencia de las observaciones es un paso esencial en la exploracin de datos y en la construccin de modelos estadsticos.
