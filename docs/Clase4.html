Clase 3: Suplementario - Pruebas no parametricas

Las pruebas no paramtricas son un conjunto de tcnicas de prueba estadstica que se utilizan para evaluar hiptesis sobre una poblacin cuando no se tienen suficientes datos para suponer una distribucin particular de los datos. A diferencia de las pruebas paramtricas, que asumen una distribucin particular de los datos (como la distribucin normal), las pruebas no paramtricas no requieren ninguna suposicin sobre la distribucin de los datos.

Algunas pruebas no paramtricas comunes incluyen:

  • Prueba de Wilcoxon: se utiliza para comparar dos grupos de datos ordenados y se utiliza a menudo cuando los datos no siguen una distribucin normal.

  • Prueba de Kruskal-Wallis: se utiliza para comparar ms de dos grupos de datos ordenados y se utiliza a menudo cuando los datos no siguen una distribucin normal.

  • Prueba de Mann-Whitney: se utiliza para comparar dos grupos de datos no ordenados y se utiliza a menudo cuando los datos no siguen una distribucin normal.

  • Prueba de chi-cuadrado: se utiliza para comparar una distribucin observada de datos con una distribucin esperada y se utiliza a menudo para comparar datos categricos.

Es importante tener en cuenta que las pruebas no paramtricas suelen ser menos robustas (es decir, tienen menor capacidad para detectar diferencias significativas) que las pruebas paramtricas, por lo que es importante elegir la prueba adecuada para su conjunto de datos y objetivo de investigacin.

Pruebas no parametricas

Prueba de Wilcoxon

La prueba de Wilcoxon es una prueba no paramtrica utilizada para comparar dos grupos de datos ordenados y se utiliza a menudo cuando los datos no siguen una distribucin normal. Esta prueba es similar a la prueba t de Student, que es una prueba paramtrica utilizada para comparar dos grupos de datos, pero la prueba de Wilcoxon no requiere que los datos sigan una distribucin normal.

Ejemplo de prueba Wilcoxon

Primero necesitamos cargar la base de datos, en este caso utlizaremos delomys.csv

Delomys <- read.csv("data/delomys.csv")

Ahora podemos realizar la prueba con el comando wilcox.test()

wilcox.test(body_mass ~ sex, data= Delomys, alternative = "two.sided")
## 
##  Wilcoxon rank sum test with continuity correction
## 
## data:  body_mass by sex
## W = 276473, p-value = 8.748e-05
## alternative hypothesis: true location shift is not equal to 0

Prueba de Mann-Whitney

La prueba de Mann-Whitney es una prueba no paramtrica utilizada para comparar dos grupos de datos no ordenados y se utiliza a menudo cuando los datos no siguen una distribucin normal. Esta prueba es similar a la prueba t de Student, que es una prueba paramtrica utilizada para comparar dos grupos de datos, pero la prueba de Mann-Whitney no requiere que los datos sigan una distribucin normal.

Es importante tener en cuenta que la prueba de Mann-Whitney solo puede utilizarse para comparar dos grupos de datos y no es adecuada para comparar ms de dos grupos. En su lugar, se puede utilizar la prueba de Kruskal-Wallis para comparar ms de dos grupos de datos ordenados o la prueba de chi-cuadrado para comparar ms de dos grupos de datos categricos.

Ejemplo de la prueba Mann-Whitney

Para realizar la prueba de Mann-Whitney en R, puedes utilizar la funcin wilcox.test() con el argumento alternative = two.sided. La sintaxis para la funcin wilcox.test() es la siguiente:

wilcox.test(body_mass ~ sex, data= Delomys, alternative = "two.sided")
## 
##  Wilcoxon rank sum test with continuity correction
## 
## data:  body_mass by sex
## W = 276473, p-value = 8.748e-05
## alternative hypothesis: true location shift is not equal to 0

La prueba de Wilcoxon y la prueba de Mann-Whitney son ambas pruebas no paramtricas utilizadas para comparar dos grupos de datos, pero hay algunas diferencias clave entre ellas. En la prueba de Wilcoxon se utiliza para comparar dos grupos de datos ordenados mientras que la prueba de Mann-Whitney se utiliza para comparar dos grupos de datos no ordenados.

Prueba de Kruskal-Wallis

La prueba de Kruskal-Wallis es una prueba no paramtrica utilizada para comparar ms de dos grupos de datos ordenados y se utiliza a menudo cuando los datos no siguen una distribucin normal. Esta prueba es similar a la prueba ANOVA, que es una prueba paramtrica utilizada para comparar ms de dos grupos de datos, pero la prueba de Kruskal-Wallis no requiere que los datos sigan una distribucin normal.

Ejemplo de prueba Kruskal-wallis

Para este ejemplo utilizaremos la base de datos penguins del paquete palmerpenguins

library(palmerpenguins)
data(peguins)
## Warning in data(peguins): data set 'peguins' not found

Utilizares la prueba de Kruskal-wallis para comparar la masa del cuerpo de los pinguinos por especie.

kruskal.test(body_mass_g ~ species, data= penguins)
## 
##  Kruskal-Wallis rank sum test
## 
## data:  body_mass_g by species
## Kruskal-Wallis chi-squared = 217.6, df = 2, p-value < 2.2e-16

Como se puedo observar el resultado indica que hay diferencias significativas de la masa del cuerpo por especie de pinguino, ahora necesitaremos realizar una prueba de contraste. En este caso utilizaremos la prueba Dunn.

La prueba de Dunn es una prueba no paramtrica utilizada para comparar ms de dos grupos de datos ordenados. Esta prueba se utiliza a menudo despus de realizar una prueba de Kruskal-Wallis, que es otra prueba no paramtrica utilizada para comparar ms de dos grupos de datos ordenados, cuando se ha encontrado una diferencia significativa entre al menos dos de los grupos.

Prueba de chi-cuadrado

La prueba de chi cuadrado es una prueba estadstica utilizada para comparar dos o ms grupos de datos categricos. Esta prueba se basa en la hiptesis nula de que no hay diferencias significativas entre los grupos y se utiliza para determinar si hay una asociacin entre las variables categricas.

Para realizar una prueba de chi cuadrado, se crea una tabla de contingencia que muestra la frecuencia de cada combinacin de categoras entre las dos variables.

Ejemplo Chi Cuadrado

Para este ejemplo utilizares la base de datos Delomys, analisaremos si existe una asiciacion entre la variable status y age.

  • Primero creamos nuestra tabla de contingencia
TablaCon <- table(Delomys$status, Delomys$age)
TablaCon
##        
##         adult juvenile
##   alive  1075      123
##   dead     60       12
  • Ahora podemos realizar la prueba de Chi Cuadrado
chisq.test(TablaCon)
## 
##  Pearson's Chi-squared test with Yates' continuity correction
## 
## data:  TablaCon
## X-squared = 2.2931, df = 1, p-value = 0.13
  • Visualizaremos nuestra relacion utilizando grafico de barra
barplot(TablaCon,
        main = "Grfico de chi cuadrado", 
        xlab = "Edad", 
        ylab = "Estado")

Graficos

Hay varios tipos de grficos que se pueden utilizar para mostrar diferencias significativas entre dos o ms grupos de datos. Algunos de los grficos ms comunes que se utilizan para este propsito son:

  • Grfico de barras: Este grfico muestra la frecuencia o el promedio de cada grupo en forma de barras verticales o horizontal. Es adecuado para comparar dos o ms grupos de datos numricos o categricos.

  • Grfico de lneas: Este grfico muestra el cambio en una variable a lo largo del tiempo o en funcin de otra variable. Es adecuado para comparar dos o ms grupos de datos numricos que tienen un orden natural.

  • Grfico de puntos: Este grfico muestra los valores individuales de cada grupo como puntos en un eje coordenado. Es adecuado para comparar dos o ms grupos de datos numricos.

  • Grfico de cajas: Este grfico muestra la distribucin de los datos de cada grupo mediante el uso de una caja que representa el rango intercuartil y dos bigotes que representan el resto de los datos. Es adecuado para comparar dos o ms grupos de datos numricos.

Es importante tener en cuenta que cada grfico tiene sus propias ventajas y desventajas y que es importante elegir el grfico adecuado en funcin de los datos y el mensaje que se desea transmitir.
